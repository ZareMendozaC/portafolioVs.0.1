About this website

Ideation & behind the scenes:

Designers and developers always tend to procrastinate projects we feel are of less importance and creating our own website tops the list. Although sometimes it is purely our workaholic tendencies that keep us busy on a project or two, we never feel the need to give that much time to our own website.

When I decided to design and develop my website I had to make sure that I give it ample of time and dedication as I didn’t wanted to end up being a part of "The cobbler's children have no shoes" phenomena.

Technical Stuff:

This website is a responsive website hand-crafted from the ground up, built on the Bootstrap framework using HTML5, CSS3 & the jQuery library. I am using the jQuery Address plugin for deep linking, Owl Carousel for sliders, jQuery easing library for the cool transitions throughout the website. Viewport element reveal transitions are done using scrollReveal.js. Pictographic scalable vector icons are from the Font Awesome library. Fonts used on the website are Raleway & Lato served by Google Fonts.

All externally sourced images used on this website are licensed under Creative Commons Zero - (CC0) by their respective owners.

No pixels were harmed in the making of this website.
